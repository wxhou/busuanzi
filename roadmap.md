# 人生K线图 - 产品路线图

## 项目愿景
用"投资视角看人生"——把八字命理的人生起伏用金融K线图的方式展示，是本产品的核心差异化卖点。

## 技术规划

- [ ] 项目技术栈从 React + Vite 迁移到 Next.js
- [ ] 理由：Next.js 提供更好的 SEO、SSR、路由管理，适合内容型和应用型混合的场景
